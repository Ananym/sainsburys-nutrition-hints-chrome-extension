Privacy Policy for Sainsbury's Nutrition Hints

Last updated: 9 March 2026

Overview

Sainsbury's Nutrition Hints does not collect, store, or transmit any personal data. The extension operates entirely within your browser.

Information Collection

This extension does not collect any personal information or sensitive user data. It does not gather your name, email address, browsing history, or any other identifiable information. No data is sent to any external server.

Use of Information

Since no user data is collected, no data is used, processed, or analysed. The extension reads nutritional information already displayed on Sainsbury's grocery pages and adds visual hints directly in your browser. All processing happens locally.

Data Storage

The extension uses Chrome's Storage API solely to cache a bundled nutritional dataset and your display preferences. This data is never transmitted to any external server and is removed when you uninstall the extension. No personal data is stored, either locally or remotely.

Data Sharing

No user data is shared with, sold to, or transferred to any third parties.

Permissions

The extension requests the following permissions:

  • Host permission for sainsburys.co.uk — used solely to inject nutrition hints into product pages. No data from these pages is collected or transmitted.
  • Storage — used to cache nutritional data locally so it doesn't need to be re-downloaded on every page load.
  • Alarms — used to schedule periodic checks for updated nutritional data.

Third-Party Services

This extension does not use any third-party analytics, advertising, tracking, or data-collection services.

Changes to This Policy

If this Privacy Policy is updated, changes will be posted on this page with a revised date.

Contact

If you have questions about this Privacy Policy, please open an issue at github.com/Ananym/sainsburys-nutrition-hints-chrome-extension.
